Create `app/memory/[id].tsx` as a new file. This screen is a modal that displays the full detail of a single memory. It inherits `presentation: "modal"` from `app/memory/_layout.tsx` automatically.

**Imports:**
- `useLocalSearchParams`, `router`, `Stack` from `expo-router`
- `useSafeAreaInsets` from `react-native-safe-area-context`
- `useMemory`, `usePerson` from `@/hooks`
- `emotionEmojis`, `formatEmotionLabel`, `formatRelativeDate` from `@/lib/formatters`
- `colors`, `fonts` from `@design/tokens`
- `View`, `Text`, `ScrollView`, `Image`, `Pressable`, `ActivityIndicator` from `react-native`
- `LinearGradient` from `expo-linear-gradient`
- `ChevronLeft` from `lucide-react-native` (used across the project for back buttons)

**Component structure:**

```typescript
export default function MemoryDetailScreen() {
  const { id } = useLocalSearchParams<{ id: string }>();
  const insets = useSafeAreaInsets();
  const { memory, isLoading } = useMemory(id ?? "");
  const { person } = usePerson(memory?.person_id ?? "");

  const handleBack = () => {
    if (router.canGoBack()) {
      router.back();
    } else {
      router.replace("/(tabs)/people");
    }
  };
```

**Screen options:** Set `headerShown: false` via `<Stack.Screen options={{ headerShown: false }} />` inside the component's return. The screen renders its own custom header with a back button, matching the style used in `app/person/[id].tsx`.

**Loading state:** While `isLoading` is true, show an `ActivityIndicator` centered on a cream background.

**Not-found state:** If `!isLoading && !memory`, call `handleBack()` and return null.

**Layout (from top to bottom):**

1. **Photo header (height: 220):** If `memory.photo_url` exists, render `<Image source={{ uri: memory.photo_url }} style={{ height: 220, width: "100%" }} resizeMode="cover" />`. Otherwise render `<LinearGradient colors={[colors.sagePale, colors.sageLight + "88"]} style={{ height: 220 }} />`.

2. **Back button overlay:** Positioned absolutely over the photo/gradient header area, top-left with safe area inset padding. White circular background (width: 36, height: 36, borderRadius: 18, backgroundColor: "rgba(255,255,255,0.9)"), with `ChevronLeft` icon inside. `onPress={handleBack}`.

3. **Content area (padding: 24):**
   - **Person name** (if person loaded): Subtle text showing "A memory with {person.name}" in `fonts.sansMedium`, `fontSize: 14`, `color: colors.warmGray`, `marginBottom: 8`.
   - **Emotion chip** (if `memory.emotion` is not null): Row with emoji from `emotionEmojis[memory.emotion]` (fontSize: 18) and label from `formatEmotionLabel(memory.emotion)` in `fonts.sansMedium`, `color: colors.warmGray`. `flexDirection: "row"`, `alignItems: "center"`, `gap: 6`, `marginBottom: 12`.
   - **Date:** `formatRelativeDate(memory.created_at)` in `fonts.sans`, `fontSize: 13`, `color: colors.warmGray`, `marginBottom: 16`.
   - **Full content text:** `memory.content` in `fonts.sans`, `fontSize: 16`, `color: colors.nearBlack`, `lineHeight: 24`. **NO `numberOfLines` prop** — the entire point of this screen is to show full content without truncation.

The whole content area is wrapped in a `ScrollView` with `style={{ flex: 1, backgroundColor: colors.cream }}` and `contentContainerStyle={{ paddingBottom: insets.bottom + 40 }}`.

**Important patterns to follow:**
- All hooks (`useMemory`, `usePerson`) must be called BEFORE any early returns (hooks before returns rule).
- Use `router.canGoBack()` check before `router.back()` — fallback to `router.replace("/(tabs)/people")`.
- Do NOT add a `<Stack.Screen name="[id]" />` registration in `_layout.tsx` — Expo Router's file convention handles it.
